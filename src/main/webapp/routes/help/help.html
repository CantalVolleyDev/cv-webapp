Pour se connecter à votre compte sur CantalVolley.fr, sélectionnez l'icône Mon Compte (affiché dans la capture présente ci-contre), pour arriver sur la page de connexion. Cette icône est accessible dans la barre de navigation principale du site (partie supérieure).

Si vous êtes un participant du championnat 4x4 :

  • Si vous êtiez inscrit au championnat l'an dernier, et que vous aviez un compte, votre adresse e-mail et votre mot de passe restent inchangés
  • Si vous faites parti d'une nouvelle équipe / ou équipe de l'an dernier et que vous avez fourni votre adresse e-mail lors des inscriptions, vous recevrez un e-mail avec vos identifiants en début de saison
  • Si vous êtes inscrit dans une équipe sans avoir fourni d'adresse e-mail, vous n'avez pas de compte officiel. Vous pouvez transmettre une demande à l'adresse jtouzy.avb@gmail.com pour obtenir un compte pour vous connecter, simplement en fournissant une adresse e-mail. Vous recevrez ensuite vos identifiants de connexion par e-mail lorsque votre demande sera prise en compte.

Mon compte

Après avoir suivi la procédure de connexion, vous arriverez sur la page associée à votre compte. Vous pouvez ainsi consulter vos différents matchs en attente.
  • Matchs en attente de validation : Ce sont les matchs que vous avez joué et dont le score a été saisi par l'équipe adverse. Vous devez valider le score pour qu'il soit pris en compte dans le classement. Pour valider le score d'un match, cliquez simplement sur le match désiré et renseignez les informations.
  • Matchs en retard : Ce sont les matchs dont la date est dépassée par rapport à la date du jour. Aucune équipe n'as encore saisi le score. Pour saisir le score d'un match, cliquez simplement sur le match désiré et renseignez les informations.
  • Prochains matchs : Dans cette liste sont présents tous les matchs qui doivent être joués par votre équipe, du plus proche au plus éloigné (par date). Pour saisir le score d'un match, cliquez simplement sur le match désiré et renseignez les informations.
  • Matchs saisis en attente : Ce sont les matchs que vous avez saisi, mais l'équipe adverse n'as pas encore validé le score. En cliquant sur un match, vous pourrez consulter la fiche complète du match.

Saisie d'un score

La saisie d'un score contient 4 étapes principales :
  • Saisir le score général : Le score général du match (par exemple : 3-1).
  • Saisir le détail du score : Le détail du score représente le score de chaque set disputé pendant le match. Des contrôles seront effectués à la validation pour vérifier la cohérence de votre saisie.
  • Saisir les joueurs du match de votre équipe : Vous devez ensuite sélectionner, dans la liste proposée, tous les joueurs ayant participer le jour du match. Ils seront ensuite visible sur la feuille de match. En fonction de l'inscription de votre équipe (équipe jouant à 3 ou 4), un minimum de joueur participant vous sera demandé.
  • Saisie d'un commentaire (facultatif) : Vous pouvez saisir un commentaire global sur le match, sur n'importe quel sujet (blessés pendant le match, casse-croûte, ambiance...). Il sera ensuite visible, ainsi que celui de l'équipe adverse, sur la feuille de match.

La validation s'effectue grâce au bouton prévu à cet effet. Après cette validation, l'équipe adverse devra valider le score pour qu'il soit pris en compte dans le classement.
