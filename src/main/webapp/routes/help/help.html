Pour se connecter à votre compte sur CantalVolley.fr, sélectionnez l'icône Mon Compte (affiché dans la capture présente ci-contre), pour arriver sur la page de connexion. Cette icône est accessible dans la barre de navigation principale du site (partie supérieure).
Si vous êtes un participant du championnat 4x4, vos identifiants vous seront transmis par e-mail, pour ceux qui l'auront fourni au moment de l'inscription. Pour les autres, il vous suffira de créer un compte sur le site avec votre adresse e-mail, et de demander le rattachement à un joueur du championnat.
